Pseudo კლასები CSS-ში გამოიყენება იმისათვის, რომ განვსაზღვროთ სტილები ელემენტის კონკრეტული მდგომარეობებისთვის, რომლებიც არ არის გამოხატული ჩვეულებრივი კლასებით ან იდენტიფიკატორებით. ისინი გვაძლევენ საშუალებას გამოვიყენოთ სტილები, რომლებიც აქტიურდება მხოლოდ მაშინ, როცა ელემენტს რაღაც სპეციფიკური მოხდება — მაგალითად, მაუსის გადატარება ან ფოკუსირება.

hover - როცა მომხმარებელი მაუსს ატარებს HTML ელემენტზე (მაგალითად, ღილაკზე ან ბმულზე).

active - როცა ელემენტი აქტიური ხდება, ანუ მაგალითად, ღილაკს დააჭერთ და ჯერ კიდევ დაჭერილია.

focus - როცა ელემენტი ფოკუსშია — ანუ მასზეა გადატანილი ყურადღება, მაგალითად კლავიატურით გადაადგილებისას ან input ველში დაწკაპუნებისას.
