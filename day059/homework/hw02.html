CSS Transition მაგალითი

რა არის Transition?

Transition იძლევა საშუალებას ელემენტის თვისებების ცვლილება მოხდეს გლუვად, ნაცვლად მყისიერი ცვლილებისა.

Transition-ის ქვეკუთვნილებები:

transition-property: განსაზღვრავს, რომელი CSS თვისება უნდა შეიცვალოს გლუვად (მაგალითად, background-color, transform).

transition-duration: განსაზღვრავს ცვლილების ხანგრძლივობას (მაგალითად, 0.4s).

transition-timing-function: განსაზღვრავს ცვლილების სიჩქარის ტიპს დროში (მაგალითად, ease-in-out).

საბოლოო შედეგი:

მაუსის მიყვანისას ღილაკი ფერს შეიცვლის და ზომაშიც გაიზრდება გლუვად და ელეგანტურად!
